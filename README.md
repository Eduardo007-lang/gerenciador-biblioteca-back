# 📚 Sistema de Gerenciamento de Biblioteca

Este é um sistema desenvolvido como parte de um teste técnico para vaga de Desenvolvedor Backend Pleno. O sistema permite o gerenciamento de usuários, livros, gêneros e empréstimos de uma biblioteca.

---

## 🚀 Tecnologias Utilizadas

### Backend
- Laravel 10+
- PHP 8.1+
- MySQL
- Laravel Sanctum (autenticação)
- L5-Swagger (documentação de API)
- API Cache com Redis (opcional)

### Frontend (opcional)
- React com Vite
- Axios
- Bootstrap (ou Tailwind CSS)
